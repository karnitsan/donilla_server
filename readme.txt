ישראל לוין: מתקין פייתון 3.
מסדר סביבה וירטואלית
python3 -m venv venv
מפעיל סביבה וירטואלית.
. venv/bin/activate
מתקין חבילות נדרשות.
pip install -r requirements.txt
מריץ סקריפט.
./deploy.sh d r
די בונה דאטהבייס (שימושי אם אתה רוצה למחוק אותו ולעשות חדש וכשהוא משתנה).
אר מריץ סרבר.
יש גם טי שמריץ בדיקות ולינטרים אם יש לך.
בפעמים הבאות אתה רק מפעיל סביבה וירטואלית ומריץ עם אר.
